# WiFiNet
# 📶 WiFiNet — Smart Fiber Internet Booking & Customer Management Portal

**WiFiNet** is a premium, single‑page Fiber Broadband Booking Portal built with a **Translucent Glassmorphism + Bento Grid UI**. It lets customers browse plans, check availability, book connections, track installations, manage subscriptions, pay bills, and download receipts — all through a sleek, modern interface.

---

## ✨ Features

- **🎨 Premium Dark UI** — Aurora gradient background, frosted glass cards, bento grid layout.
- **🔐 Secure Authentication** — Login/Register with demo credentials. Role‑aware navigation.
- **🏠 Home Page** — Speed statistics, testimonials, quick action buttons.
- **📡 Broadband Plans** — Five tiered plans (100 Mbps – 1 Gbps) with OTT benefits and instant booking.
- **📍 Fiber Availability Checker** — Validate serviceability by PIN code.
- **📋 Connection Booking** — Capture customer details, plan selection, installation date/time.
- **🔧 Installation Tracker** — Animated timeline (Booking → Verification → Engineer → Installation → Activated).
- **💳 Payment Portal** — Demo payment with receipt generation.
- **📊 Usage Analytics** — Canvas‑drawn bar chart of monthly data consumption.
- **🎧 Support Center** — Raise tickets (Billing, Speed, Installation) and view ticket history.
- **👤 Dashboard** — Active plan, connection status, quick actions, and renewal info.
- **📱 Fully Responsive** — Works on desktop, tablet, and mobile.
- **💾 Local Storage Persistence** — All bookings, tickets, and connections survive page reloads.
